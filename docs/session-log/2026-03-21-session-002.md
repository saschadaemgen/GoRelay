### Architecture Rules (Non-Negotiable)

- Never log IP addresses, queue IDs, or message content
- Never use em dashes
- Never change version numbers without permission
- Three goroutines per connection (receiver, processor, sender)
- io.ReadFull for all reads, never plain conn.Read
- Fixed 16,384 byte blocks, always
- All command handlers must be idempotent
